# Photo management

A Scripting app script for managing photos with a card-based workflow.

## Features

- Browse photos in a swipeable card interface.
- Right swipe or tap delete to add photos to a pending delete queue.
- Confirm deletion through the iOS Photos system prompt.
- Left swipe to skip without marking a photo as processed.
- Add photos to user albums and automatically advance to the next photo.
- Persist processed photo IDs locally so handled photos do not reappear.
- Manage album order with a native editable list.

## Privacy

This repository contains only the script source code. It does not include photos, photo identifiers, local storage data, tokens, or personal account credentials.
